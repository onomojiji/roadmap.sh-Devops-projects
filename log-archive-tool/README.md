## Roadmap.sh-Log-Archive-Tool

<https://roadmap.sh/projects/log-archive-tool>

> Keys steps to archive logs in your system

To run this bash code you need to follow theses steps.

### 1. Clone the repository
````
git clone git@github.com:onomojiji/roadmap.sh-Devops-projects.git
````

### 2. Navigate to the script folder
````
cd log-archive-tool
````

### 3. Give execution permissions to the file
````
chmod +x log-archive-tool.sh
````
Your terminal most return a least of this permission
````
-rwxr-xr-x 1 group user 2063 month.  day time log-archive-tool.sh
````

### 4. If you want you can edit the script
````
sudo vim log-archive-tool.sh
````

### 5. When every thing is good for you run the code
````
./log-archive-tool.sh
````
