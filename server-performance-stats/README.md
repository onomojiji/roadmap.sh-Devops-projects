## Roadmap.sh-SSH-Server-Performance-Stats

<https://roadmap.sh/projects/server-stats>

> Keys steps to display your Server performance stats

To run this bash code you need to follow theses steps.

### 1. Clone the repository
````
git clone git@github.com:onomojiji/roadmap.sh-Devops-projects.git
````

### 2. Navigate to the script folder
````
cd server-performance-stats
````

### 3. Give execution permissions to the file
````
chmod +x server-performance-stats.sh
````
Your terminal most return a least of this permission
````
-rwxr-xr-x 1 group user 2063 month.  day time server-performance-stats.sh
````

### 4. If you want you can edit the script
````
sudo vim server-performance-stats.sh
````

### 5. When every thing is good for you run the code
````
./server-performance-stats.sh
````
